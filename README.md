# 🧠 AI Topic Explainer

Explain any topic in seconds with AI-powered structured output.

🚀 Built using **Streamlit + Groq**

---

## ✨ Features

- ⚡ Instant topic explanation  
- 🎯 Beginner to Advanced levels  
- 💡 Real-life examples  
- 🧠 Key bullet points  
- 🔴 Live streaming AI response  
- 🎨 Glassmorphism premium UI  
- 📥 Download notes  
- 📋 Copy-friendly output

## 🛠 Tech Stack

- Python  
- Streamlit  
- Groq API  
- LLaMA 3.1  
- HTML/CSS
